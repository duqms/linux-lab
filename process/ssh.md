* ssh 포트 변경

  /etc/ssh/sshd_config  파일 Port 수정 후 sshd 서비스 재기동
------------------------------------------------------------------------------------------------------
* ssh 디버그 모드 로그인

  ssh -vvv -p2222 user@172.0.0.1
------------------------------------------------------------------------------------------------------
* known hosts 에 등록' 후  자동 로그인 등록 필요시

  ssh-copy-id -i ~/.ssh/id_rsa.pub user@172.0.0.1
------------------------------------------------------------------------------------------------------
* 'known_hosts에 키 값 재등록을 위해'서 key 값을 삭제 할때

  ssh-keygen -f "/home/user/.ssh/known_hosts" -R "[192.168.0.1]:2222"
